# Portfolio

A personal portfolio built with React, Vite, and Resend to send emails.

## Features

* Hero section with a greeting and a call to action to download my CV.
* Skills section with a matrix of technical and soft skills.
* Timeline section with a list of milestones.
* Projects section with a list of projects with code and page URLs.
* Contact footer with a form to send emails.

## Technologies Used

* React for building the user interface.
* Vite for building and serving the application.
* Resend for sending emails.

## How to Run

1. Clone the repository.
2. Run `npm install` to install the dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:5173` in your web browser.
